Yes. And you don't have to prove your hidden statement (which is a consequence of Fermat's little theorem). That all rationals form periodic decimals follows from the method for finding the decimal expression:
 
Consider the rational number N/M. To get the integer part of the expression, we write N = d0M + q0, where d0 is the integer portion, and q0 is the remainder, and therefore < M. The next digit, d1, is the result of the division 10q0 = d1M + q1. Since 10q0 < 10M, d1 is a single digit integer. We continue adding digits in this fashion: 10qn-1 = dnM+qn for all n.
 
The key to note here is that all of the qn < M. Since they are all non-negative integers, there are only a finite number of values they can be, but there are an infinite number of values of n. Eventually, for some n, qn is going to take on the same value it did before. From this point on, all the divisions to determine new digits will be repeats of the earlier divisions, so the new digits will be the same as before.
 
 
For example, with 5/7:
 
A. 5 = 0*7 + 5    decimal: 0.
B. 50 = 7*7 + 1  decimal: 0.7
C. 10 = 1*7 + 3  decimal: 0.71
D. 30 = 4*7 + 2  decimal: 0.714
E. 20 = 2*7 + 6  decimal: 0.7142
F. 60 = 8*7 + 4  decimal: 0.71428
G. 40 = 5*7 + 5  decimal: 0.714285  Note that the remainder repeats A.
H. 50 = 7*7 + 1  decimal: 0.7142857     Repeats B
I. 10 = 1*7 + 3  decimal: 0.71428571    Repeats C
J. 30 = 4*7 + 2  decimal: 0.714285714  Repeats D
etc.
